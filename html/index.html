PORTFOLIO

Organizr

Organizr is a Spring Boot project designed to match patients in need of organ transplants with compatible donors. The app uses a REST API to consume and parse data that is persisted in a multi table Realational Database. Database transactions are done using SpringDataJPA. Front end design was done using Angular. This project was built under Agile Methodologie in a one week sprint with regular Scrum stand ups.

GameBored

GameBored is a social media site designed to connect people throught their love of board games. This is a Spring Boot project using Spring MVC, JPA, JSTL, with a multi table MySQL database. This project was built under Agile Methodologie in a one week sprint with regular Scrum stand ups. The app is deployed using AWS and Apache Tomcat for the server.
